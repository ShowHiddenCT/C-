# 一些函数
**(加#include< cmath>)(一个函数库)**
* pow(x,y) 求给定y次幂的x。
* cbrt(x)  计算立方根。
* hypot(x,y) 计算两个给定数的平方和**的平方根**。
* sqrt(x) 计算复平方根
 ## sqrt
 1. 参数
 * 浮点数或整形
 2. 返回值
 * 无错误就返回平方根。
 * 如定义域错误，就返回定义值(NaN).
 * 如下溢导致值域错误，就返回**舍入后**的结果。
 3. example
 * sqrt(100)  //10
 * sqrt(2)   // 1.41421
 * sqrt(0)  //0
 * sqrt(-1) //nan
